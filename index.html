Projects

FabFlix Website

FabFlix is a movie website I developed as part of my CS122B course. It allows users to explore movies through features like fuzzy search, genre browsing, and searching by star or movie name. The project was built on AWS and involved extensive use of HTML, CSS, JavaScript, and Java web services. Developing FabFlix was challenging yet rewarding, as it required working across a broad spectrum of technologies—from front-end web design to back-end services—giving me valuable experience in full-stack development.
